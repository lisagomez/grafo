# global/ — Conocimiento transversal

Conocimiento **independiente de jurisdicción**: entidades del dominio, flujos de decisión,
conceptos y definiciones compartidas por todos los países.

Lo específico de un país vive en `countries/<ISO-2>/`. Aquí solo va lo que no depende de
una legislación concreta. Mantener consistente con `.claude/memory/BUSINESS_LOGIC.md`
(§2 Entidades, §3 Flujos).
