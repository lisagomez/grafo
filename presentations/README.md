# Presentaciones — Grafo

Carpeta para las presentaciones del proyecto (pitch a clientes, demos, material comercial).

## Índice

| Archivo | Audiencia | Descripción |
|---|---|---|
| [`grafo-presentacion-cliente.md`](grafo-presentacion-cliente.md) | Cliente / Despacho contable | Pitch comercial: problema, solución, diferenciadores, valor. |

## Formato

Los decks están en **Markdown estilo [Marp](https://marp.app/)** (cada `---` es una diapositiva).
- Se leen tal cual como Markdown.
- Para exportar a slides (PDF/PPTX/HTML): instalar Marp (`npx @marp-team/marp-cli@latest grafo-presentacion-cliente.md --pdf`) o la extensión "Marp for VS Code".

> Fuente de verdad del contenido: `.claude/memory/BUSINESS_LOGIC.md`. Mantener el pitch alineado con el dominio.
